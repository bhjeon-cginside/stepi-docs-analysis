# STEPI 간행물 분석 개요

STEPI(과학기술정책연구원) 40년 기관 생산 발간물의 문서 형식, 연도별/문서유형별 수량, PDF 페이지 규모, PDF 파싱 난이도 확인용 다중 페이지 캡처를 정리한 정적 보고서입니다.

## GitHub Pages

정적 사이트는 `docs/index.html`에 있습니다.

GitHub 저장소 설정에서:

1. **Settings → Pages**
2. **Build and deployment → Source: Deploy from a branch**
3. **Branch: main / folder: /docs**
4. Save

으로 설정하면 GitHub Pages로 공개됩니다.

## 주요 산출물

- `docs/index.html`: GitHub Pages용 웹 보고서
- `docs/assets/pdf_captures_multi/`: 대표 PDF 다중 페이지 캡처 이미지
- `stepi_publication_analysis_overview.md`: Markdown 개요
- `document_inventory_report.md`: 전체 문서 현황 보고서
- `document_inventory_files.csv`: 전체 파일 목록
- `pdf_page_count_summary.json`: PDF 페이지 수 산출 결과

## 원자료 제외

원본 tar 및 해제된 전체 문서 원문은 용량과 저작권/배포 범위 이슈로 Git 저장소에 포함하지 않습니다.
